# Сегментация людей и дорожных ям

Хоть и не было тз под эту вакансию но мне задача сегментации показалась очень интересной.
Я решил сделать небольшое приложение для сегмантции людей и дорожных ям
Дообучил готовую модель YOLO на открытом датасете для ям, а для людей подходит дефолтная yolo26n потому что она уже хорошо распознаёт людей 


## ВАЖНО: Первый запуск модели может происходит долго (минуты) Потому что загружаются веса для людей из интернета

## Как обучалась модель ям

Для обучения использовал [Pothole Image Segmentation Dataset](https://www.kaggle.com/datasets/raunakkesharwani/pothole-image-segmentation-dataset). Обучение запускал в Google Colab на Tesla T4. В тренировочную часть вошло 1590 изображений, в проверочную — 381.

Основные параметры запуска:

| Параметр | Значение |
|---|---:|
| Исходная модель | `yolo26n-seg.pt` |
| Эпохи | 50 |
| Размер изображения | 640 × 640 |
| Batch | автоматический, фактически 16 |
| Оптимизатор | AdamW |
| Начальная скорость обучения | 0.002 |
| Momentum | 0.9 |
| Patience | 10 |
| Workers | 2 |
| Seed | 42 |

Лучший результат получился примерно на 48-й эпохе:

| Метрика | Результат |
|---|---:|
| Mask Precision | 78.4% |
| Mask Recall | 68.0% |
| Mask mAP50 | 75.5% |
| Mask mAP50-95 | 45.8% |

Это показатели на проверочной части конкретного датасета

## Как работает приложение

1. Пользователь загружает фотографию или делает снимок с камеры.
2. Дообученная модель ищет ямы и строит для них маски.
3. Готовая модель запускается только для класса `person`.
4. Результаты двух моделей накладываются на исходное изображение.
5. Интерфейс показывает количество объектов и время обработки.

Оранжевым отмечаются ямы, синим — люди. Порог уверенности можно менять в боковой панели.

## Запуск на Windows

Сначала откройте командную строку в папке проекта и создайте виртуальное окружение:

```bat
py -3.11 -m venv .venv
.venv\Scripts\activate
```

Установите зависимости:

```bat
python -m pip install --upgrade pip
pip install -r requirements.txt
```

Запустите приложение:

```bat
streamlit run app.py
```

После запуска страница откроется в браузере. При первом старте нужен интернет: Ultralytics скачает стандартную модель для поиска людей. В дальнейшем она будет использоваться из локального кэша.

## Ограничения

Модель ям обучалась на готовом датасете, поэтому результат зависит от того, насколько новая фотография похожа на его изображения. Сложнее всего могут обрабатываться ночные кадры, снег, сильные блики, мокрый асфальт и очень маленькие ямы. Для полноценной проверки на российских дорогах нужен отдельный набор местных фотографий и дополнительная разметка.
